# 权威动态

本板块用于沉淀“每月一次”的数字法治权威动态月报，覆盖政策监管、司法动态与学术研究三类来源。

## 收录范围

- 国家机关与监管部门官网公开信息（如中国政府网、国家网信办、最高人民法院）
- 权威学术机构公开信息（如中国社会科学网法学频道）
- 以“标题、发布日期、来源链接、要点摘要”为统一结构

## 更新频率

- 固定每月运行一次月报生成任务
- 月报命名规则：`authority-brief-YYYY-MM.md`
